Enables comment notifications

== Description ==
This plugin allows you to define the e-mails sent when someone adds a comment to a post.<br>

E-mails can be sent to the author of the post or, in case of a reply, to the comment write to which the coment is addressed

This can be adjusted granular: comments can be enable on a per post type basis and can be disabled for specific posts.
